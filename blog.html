Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element in the DOM, like getElementById, getElementsByClassName, querySelector, and querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The main difference between innerText, innerHTML, and textContent is: innerText returns the text inside an element, innerHTML returns the full HTML structure of the particular element, and textContent is like innerText but doesn't consider the element's styling, like display: hidden;

Question-3: What is event delegation in the DOM?

Event delegation allows us to bubbling an event listener to all child elements by adding just one event listener to the parent. This makes our code run more efficiently and faster.

Question-4: What is event bubbling in the DOM?

Event bubbling is... when an event listener is triggered on an element, and it doesn't stay only on that element. It also triggers its parent element, then the parent's parent, and continues going up until we stop it manually.

Question-5: How do you create, add, and remove elements using JavaScript?

for create an element I use document.createElement("tagName");
for add an element I use parentName.appendChild(element);
for remove an element I use parent.removeChild(element);
